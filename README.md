# Dating Chat Coach

> Short description: 分析女生情绪信号，指导男生用自然、不施压的方式把握回复时机和聊天表达。

`dating-chat-coach` 是一个面向男性用户的恋爱聊天教练 skill，重点帮助你判断女生当下的情绪和意图，决定要不要立刻回复，并把原本想发的话优化成更自然、没有 AI 味的日常表达。

它主要解决三件事：

- 看懂女生最新消息更可能在表达什么
- 判断这轮是该现在回、晚点回，还是先别回
- 把男生原本想发的话改成更短、更自然、更像真人聊天

它还包含高风险识别能力，可识别明显低兴趣、索要礼物、转账、红包、消费证明诚意等风险场景。

## Description

这个 skill 适用于这样的场景：男生对女生有好感，但不太会聊天，容易误判女生的情绪，也不知道怎样回复才更自然。

它会重点帮助你：

- 分析 `嗯`、`哦`、`没事`、`你忙吧` 这类简短或高歧义回复中的情绪线索
- 判断女生更可能是在想要空间、礼貌收尾、想被理解，还是已经明显兴趣不高
- 避免男生常见失误，比如解释过多、连发消息、太舔、太油、太像 AI
- 在不适合立刻回复时，给出更真实的等待时长建议
- 在出现低兴趣、物质试探、转账红包、礼物索取等情况时给出明确预警

这个 skill 不用于 PUA、操控、施压，也不鼓励通过金钱、礼物或反复讨好来推进关系。

## User Input

为了让判断更准，建议尽量提供完整上下文。

推荐输入格式：

```text
女生最新消息:
最近3-10轮聊天记录:
你原本想发的话: (optional)
关系阶段: 刚认识 / 刚开始聊天 / 有点熟 / 暧昧试探 / 快冷掉了 / 关系不明 / 疑似被吊着
敏感节点: 无 / 见面 / 表白 / 吵架 / 冷战 / 红包/转账 / 礼物 / 借钱 / 消费证明诚意
```

最少建议提供：

- 女生最新消息
- 最近几轮聊天记录

可选但很有帮助的信息：

- 你原本想发的话
- 当前是刚认识、暧昧试探、还是快冷掉的阶段
- 是否涉及见面、吵架、礼物、红包、借钱等敏感节点

## Output

这个 skill 会输出结构化分析结果，固定包含以下字段：

```text
情绪判断：
潜在意图：
风险等级：
风险提醒：
风险点：
建议是否立刻回：
等待时长：
回复思路：
可直接发送的3条日常话术：
```

### 输出字段说明

- `情绪判断`: 她此刻更可能处于什么情绪状态
- `潜在意图`: 她这句话更可能想表达什么、想让你怎么接
- `风险等级`: `低 / 中 / 高 / 爆高`
- `风险提醒`: 对明显低兴趣、礼物压力、红包转账等风险的提醒
- `风险点`: 男生此刻最容易踩的坑
- `建议是否立刻回`: `现在回 / 稍后回 / 先别回`
- `等待时长`: 更自然的等待建议和原因说明
- `回复思路`: 本轮回复应该以什么目标为主
- `可直接发送的3条日常话术`: 3 条可直接发送、语气略有区别的自然回复

## Style Rules

生成的回复必须像真实聊天，而不是像 AI 写出来的标准答案。

这个 skill 会尽量避免：

- AI 味太重的表达
- 过于像心理咨询或客服的话术
- 过度打磨、过度完美、像背稿一样的回复
- 太舔、太黏、太自我感动的表达
- 土味情话和油腻撩法

例如会明确避免这类表达：

- `我在输液，输的是想你的液`
- 夸张的情绪施压
- 一看就很刻意的“高情商标准答案”

## Risk Detection

这个 skill 会在必要时把结果提升为 `高风险` 或 `爆高风险`。

典型 `高风险` 信号：

- 长期冷淡维持型回复
- 基本不接话、不推进话题
- 一直是男生单方面投入
- 明显兴趣不足但还在被动吊着

典型 `爆高风险` 信号：

- 要钱
- 要转账或红包
- 用送礼来证明诚意
- 把花钱和感情态度强绑定

出现这些情况时，skill 会优先提醒用户谨慎，而不是继续帮用户加大投入。

## Example Use

```text
女生最新消息: 嗯
最近3-10轮聊天记录:
你: 今天是不是有点累
她: 还好
你: 感觉你说话都没什么精神
她: 嗯

你原本想发的话: 是不是我惹你不开心了 你可以告诉我 我会一直陪你
关系阶段: 刚开始聊天
敏感节点: 无
```

期望输出逻辑：

- 不直接武断判断成“她不喜欢你”
- 会同时考虑疲惫、情绪低、懒得展开、轻度敷衍等几种可能
- 会避免男生发出太黏、太焦虑、太自我感动的回复
- 可能会建议先等一会儿，再用更轻一点的话重新接触
